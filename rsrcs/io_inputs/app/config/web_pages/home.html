PyScript Exercises.


This is a list of exercises to study PyScript. These exercises come from the official PyScript documentation, where several examples are listed, starting with basics such as 'Hello World', and digging into plotting data.


  • Hello World: start learing how to use PyScript to run Python in a browser.
  • Antigravity: import files, using PyScript configuration file (TOML or JSON).
