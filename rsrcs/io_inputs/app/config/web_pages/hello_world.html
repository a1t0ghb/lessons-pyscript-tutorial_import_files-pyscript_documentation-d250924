'Hello World' Example.

URL for reference: 'https://pyscript.com/@examples/hello-world/latest'.


This is a basic example of how to create a PyScript application that adds a terminal to the page and prints "Hello World" to it. This example is meant to be a starting point for creating more complex applications.


Using 'pyscript.display': default (i.e. 'append=True') vs. 'append=False'.

  • Default: encapsulates the result from 'pyscript.display' between a 'div' tag. Then the 'div' gets ADDED to the inner content of the target DOM element.
  • 'append=False': the result from 'pyscript.display' RE-WRITES WHATEVER previous inner content of the target DOM.

IMPORTANT: whenever as possible, create an html element (and include an id) to populate with the result of 'pyscript.display' by using the 'target=' parameter / property. Otherwise, un-expected behaviours can happen, as the one described in the comments of this html.


Use the 'target=' specification inside the pyscript as a parameter of the 'pyscript.display' function: 'from pyscript import display display("PyScript running...", target="pyscript_output")'.
